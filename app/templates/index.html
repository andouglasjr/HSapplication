This interface was developed by Andouglas Junior in his thesis to operate the Holographic System, but can be used in any holographic experiment that uses a CCD sensor (camera) to acquire the hologram.

The process that could be performed for this interface are:

  • Acquisition: The camera plugged in the computer is accessed by the browser and the hologram is shown in the view.
  • Processing:In this step is possible change some image properties, like bright and contrast.
  • Reconstruction: The spectrum angle method is used in order to obtain the amplitude and phase of hologram.
  • Segmentation: Depending on how the samples is disposed in the image, it is necessary to perform separation to be inseting on the machine learning model.
  • Machine Learning: In this last step, the user can upload some ML models and use it to classify the sample.

To start the process click on the button "Next" on the top-right.
